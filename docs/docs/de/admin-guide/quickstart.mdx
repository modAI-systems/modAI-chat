## Voraussetzungen

- [Docker](https://docs.docker.com/get-docker/) mit Compose

## 1. Compose-Datei herunterladen

```bash
curl -O https://raw.githubusercontent.com/modAI-systems/modAI-chat/main/resources/compose-files/compose-quickstart.yaml
```

## 2. modAI starten

```bash
docker compose -f compose-quickstart.yaml up -d
```

Öffnen Sie http://localhost — Login: `admin` / `admin`

NanoIDP-Dashboard: http://localhost:8000

## Stoppen

```bash
docker compose -f compose-quickstart.yaml down
```
